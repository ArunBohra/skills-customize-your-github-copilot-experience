# 📘 Assignment: Python Web Scraping with BeautifulSoup

## 🎯 Objective

Learn how to fetch web pages and extract structured information using Python, `requests`, and `BeautifulSoup`.

## 📝 Tasks

### 🛠️ Fetch and Parse a Web Page

#### Description

Write code to download a web page and parse its HTML so you can extract meaningful data.

#### Requirements
Completed program should:

- Use the `requests` library to fetch a web page URL
- Parse the HTML response with `BeautifulSoup`
- Find and print at least one type of element, such as headings, links, or paragraph text
- Handle request failures cleanly

### 🛠️ Extract and Save Scraped Data

#### Description

Extract a list of items from the parsed HTML and save the results in a structured format.

#### Requirements
Completed program should:

- Locate multiple elements from the page (for example, article titles or links)
- Build a list of extracted data items
- Save the scraped data to a file, such as `results.txt` or `results.csv`
- Include a short summary of how many items were extracted
